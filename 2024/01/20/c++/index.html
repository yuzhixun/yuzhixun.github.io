迭代器失效

什么叫做迭代器失效？
当我们给一个容器插入一个值以后，这个容器会发生移动，到达内存的任何位置，以至于我们的it指针会失效出现两种情况，其一就是输出任意一个值，其二就是导致系统崩掉

插入带来的迭代器失效

解决方案

​ 我们需要知道，v.insert其实是一个返回类型的操作。它会返回新的迭代器的地址，也就是当容器整体发生了偏移，而指向同一个下标的指针也会跟着发生偏移。所以我们需要一个新的迭代器来储存他的返回值。

#include <iostream>
#include <vector> 

using namespace std;

int main(){
    vector<int> v;
    v.push_back(1);
    v.push_back(2);
    v.push_back(3);
    v.push_back(4);
    
    vector<int>::iterator it = v.begin() + 3;
    it = v.insert(it,6);
    
    cout << *it << endl;
    
    return 0;
}

看14行，我们创建了一个新的迭代器来储存它的返回值。

list容器

list容器的简介

list是个双向链表，不可以随机的存取元素，不支持at.(pos)函数与【】操作符。

list构造对象

list<int> IstInt;//定义一个存放int的list容器。
list<float> IstFloat;//定义一个存放float的list容器。
list<string> IstString;//定义一个存放string的list容器。

进行操作

  • list.push_back(elem);//早容器尾部加入一个元素
  • list.pop_back();//删除容器中最后一个元素
  • list.push_front(elem);//在容器开头插入一个元素
  • list.pop_front();//删除容器中第一个元素

list容器带参数的构造方法

  • list(n,elem);//构造函数将n个elem拷贝给本身。
  • list(beg,end);//构造函数将[beg,end）区间中的元素拷贝给本身。左闭右开
  • list(const list &lst);//拷贝构造函数。

list(n,elem)：给list容器里拷贝n个elem，我们也可以说是初始化。

list(beg,end)：我们不能写成list(lst.begin()+1,lst.begin()+2);//因为list容器不支持＋n的操作

我们可以通过不断使用beg++或者end–来进行指针移动的操作

  • list.assign(beg,end);//将[beg,end)区间中的数据拷贝赋值给本身。注意该区间是左闭右开的区间。

  • list& operator = (const list &lst);//重载等号操作符。
  • list.swap(lst);//将lst与本身的元素互换。

list的插入

  • list.insert(pos,elem);//在pos位置插入一个elem元素的拷贝，返回新数据的位置。
  • list.insert(pos,n,elem);//在pos位置插入n个elem数据，无返回值。
  • list.insert(pos,beg,end);//在pos位置插入[beg,end)区间的数据，无返回值。

list的删除

  • list.clear();//移除容器里的所有数据。

  • list.erase(beg,end);//删除[beg,end）区间的数据，返回下一个数据的位置。

    • 经测试我们可以总结出他是一个过程函数，也是一个结果函数。而他的返回值就是end所指向的位置的数据。

  • list.erase(pos);//删除pos位置的数据，返回下一个数据(也就是pos+1)的位置。

  • lst.remove(elem);//删除容器中所有与elem值匹配的元素。

list的反序排列

  • lst.reverse();//反转链表，比如lst包含1，3，5元素，运行此方法后，lst就包含5，3，1元素。

set和 multiset容器

set/multiset容器简介

  • set是一个集合容器，其中所包含的元素是唯一的，集合中的元素按一定的顺序排列。元素插入的过程是按排列规则插入，所以不能跟指定插入位置。
  • set是采用红黑树（也就是二叉树）变体的数据结构实现的。再插入和删除操作上比vector快。
  • set不可直接存取元素。（不可以使用at.(pos) 与 []操作符。
  • set容器是默认升序排序

大多数情况下set都是实现一个去重的功能。

set容器插入与迭代器

  • set.insert(elem);//在容器中插入元素
  • set.begin();//返回容器中第一个数据的迭代器
  • set.end();//返回容器中最后一个数据的迭代器
  • set.rbegin();//返回容器中倒数第一个元素的迭代器
  • set.rend();//返回容器中倒数最后一个元素的后面的迭代器
#include <iostream>
#include <set>

using namespace std;

int main(){
    set<int> s1;
    set<int>::iterator it;
    s1.insert(1);
    s1.insert(5);
    s1.insert(3);
    s1.insert(7);
    s1.insert(9);
    s1.insert(2);
    
    for(it = s1.begin();it!=s1.end();it++){
        cout << *it << ' ';	
    }
    cout << endl;
    
    set<int>::reverse_iterator it1;
    for(it1 = s1.rbegin();it1!=s1.rend();it1++){
        cout << *it1 << ' ';
    }
    cout << endl;
    return 0;
}

  • 返回容器中与elem相等的上下限的两个迭代器。上限是闭区间，下限是开区间。

map容器的迭代

我们要搞清楚一个东西：存在map容器里的是整型和字符串成对的形式，所以无论是插入还是输出我们都要用pair。我们在输出的时候，为什么不直接使用“*it1.first”和“*it1.second”，因为*it是map的迭代器，没用first这个成员，所以我们在输出的时候一定要用pair类型的

#include <iostream>
#include <map>
using namespace std;

class Student{

        int id;
        string name;

        }

            this->id = id;
            this->name = name;
        }
};

int main(){

    map1.insert(pair<int,string>(3,"余志勋")); //3是键，“小张”是值

    map1.insert(pair<int,string>(5,"季兆程")); 
    
    map1.insert(map<int,string>::value_type(1,"孟慧雪"));
    map1[2] = "罗明杰";
    
    Student s1(1,"xiaoli");
    stus.insert(pair<int,Student>(s1.id,s1));
    Student s2(2,"xiaozhou");
    stus.insert(map<int,Student>::value_type(s2.id,s2));
    Student s3(3,"xiozhao");
    stus[s3.id] = s3;
    
    map<int,string>::iterator it1;
    for(it1 = map1.begin();it1!=map1.end();it1++){
        pair<int,string> p = *it1;
        int key = p.first;
        string value = p.second;
        
        cout << key << ' ' << value << endl;
