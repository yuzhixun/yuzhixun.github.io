java的基础语言类型

java是强类型的语言，所以在定义long 和 float的时候，一定要加后缀来表明类型。

注意某些类型的后缀

public class demo1 {
    public static void main(String[] args) {
        long i = 200000000000L;
        System.out.println(i);
    }
}

数据类型的扩展

八进制以0开头；

十六进制以0x开头；

最后输出以十进制输出；

public class demo1 {
    public static void main(String[] args) {
        int i = 10;
        int i1 = 010;
        int i2 = 0x10;
        
        System.out.println(i);
        System.out.println(i1);
        System.out.println(i2);
    }
}
