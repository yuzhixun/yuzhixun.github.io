持续输入多个数，求他们的总和和平均值

package com.yuzhixun.base;

import java.util.Scanner;

public class demo2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int num = 0;
        double sum = 0;

        while(sc.hasNextDouble()){
            double temp = sc.nextInt();
            num++;
            sum += temp;
        }

        System.out.println(num + "个数相加的结果为：" + sum);
        System.out.println(num + "个数相加的平均值为：" + sum/num);
        sc.close();
    }
}

可变参数

package com.yuzhixun.base;

public class demo3 {
    public static void main(String[] args) {
        printMax(34,3,3,2,56);
    }

    public static void printMax(int... numbers){
        if(numbers.length == 0){
            System.out.println("Nothing");
            return;
        }
        for(int i = 0;i < numbers.length-1;i++){
            for(int j = i+1;j < numbers.length;j++){
                if(numbers[i] > numbers[j]){
                    int temp = numbers[i];
                    numbers[i] = numbers[j];
                    numbers[j] = temp;
                }
            }
        }

        for(int a:numbers){
            System.out.println(a);
        }
    }
}

我们的方法名是printMax,我们接受的参数是并不是直接传递的数组，而是可变的数组，这里的形参int… numbers,就是接受可变的数组

冒泡排序

package com.yuzhixun.base;

import java.util.Arrays;

public class demo4 {
    public static void main(String[] args) {
        int[] nums = {1, 9, 8, 33, 65, 2, 7, 55, 69, 41, 25, 325, 987, 654};
        int[] temp = maopao(nums);
        System.out.println(Arrays.toString(temp));
    }

    public static int[] maopao(int[] arrar) {
        for (int i = 0; i < arrar.length; i++) {
            for (int j = 0; j < arrar.length - 1 - i; j++) {
                if (arrar[j + 1] < arrar[j]) {
                    int temp = arrar[j];
                    arrar[j] = arrar[j + 1];
                    arrar[j + 1] = temp;
                }
            }
        }
        return arrar;
    }
}
