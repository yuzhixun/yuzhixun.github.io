初识vue

小案例1

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>初识vue</title>
    <script type="text/javascript" src="../js/vue.js"></script>
</head>
<body>
    <div id="root">
        <h1>hello!我是{{name}}</h1>
        <h1>今年我{{age}}岁</h1>
        <h1>我最喜欢的英雄是{{hero}}</h1>
    </div>

    <script type="text/javascript">
        Vue.config.productionTip = false

        new Vue({
            el:'#root', //el用于指定当前vue实例为那个容器服务，值通常是css选择器的字符串。
            data:{
                name:'余志勋',
                age:'18',
                hero:'李白'
            }
        })
    </script>
</body>
</html>

初识vue

  1. 想让vue工作，就必须创建一个Vue实例，且要传入一个配置对象。
  2. root容器里的代码依然符合html的规格，只不过混入了一些特殊的Vue语法。
  3. root容器里的代码被称为【vue模板】
  4. 一个实例和一个vue容器是一一对应的关系。

vue的模板语法

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>初识vue</title>
    <script type="text/javascript" src="../js/vue.js"></script>
</head>
<body>
    <div id="root">
        <h1>插值语法</h1>
        <h3>你好，{{name}}</h3>
        <hr>
        <h1>指令语法</h1>
        <a v-bind:href="url">点我去小鱼丸的个人博客</a> 
        <a :href="url">点我去小鱼丸的个人博客</a>
    </div>

    <script type="text/javascript">
        Vue.config.productionTip = false

        new Vue({
            el:'#root',
            data:{
                name:'jack',
                url:'https://yuzhixun.github.io/'
            }
        })
    </script>
</body>
</html>

Vue模板语法有两大类：

  1. 插值语法：
    • 功能：用于解析标签体内容。
    • 写法：，xxx是js表达式，且可以直接读取到data中的所有属性。
  2. 指令语法
    • 功能：用于解析标签（包括：标签属性，标签体内容，绑定事件………..)。
    • 举例：v-bind:href = ‘’xxx‘’ 或 简写成 :href=“xxx”,xxx同样要简写成js表达式，且可以直接读取到data里的所有属性。
    • 备注：Vue中有很多的指令，且形式都是: v-????，此处我们只拿v-bind举个例子。

当遇到多个插值时，我们可以在Vue当中创建一个对象，xxx.XXX

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>初识vue</title>
    <script type="text/javascript" src="../js/vue.js"></script>
</head>
<body>
    <div id="root">
        <h1>插值语法</h1>
        <h3>你好，{{name}}</h3>
        <hr>
        <h1>指令语法</h1>
        <a v-bind:href="url">点我去小鱼丸的个人博客</a> 
        <a :href="school.url">点击我去{{school.name}}院的官网</a>
    </div>

    <script type="text/javascript">
        Vue.config.productionTip = false

        new Vue({
            el:'#root',
            data:{
                name:'刘泽',
                url:'https://yuzhixun.github.io/',
                school:{
                    name:'湖北警官学院',
                    url:'https://www.hbpa.edu.cn/'
                }
            }
        })
    </script>
</body>
</html>
