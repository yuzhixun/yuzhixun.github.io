    int i,n,A[255];
    n=0;
    while(!StackEmpty(S)){n++;Pop(S,A[n]);};
    for(i=1;i<=n;i++)Push(S,A[i]);

Status algo2(Stack S,int e){
    Stack T;int d;
    InitStack(T);
    while(!StackEmpty(S)){
        POp(S,d);
        if(d!=e)Push(T,d);
    }
    while(!StackEmpty(T)){
        Pop(T,d);
        Push(S,d);
    }
}

功能：先将不等于e的元素取出放入到栈T当中，然后又将T中的值返回给S当中，这样栈S就没有与e相等的元素了。

3.11

简述队列和栈这两种数据类型的相同点和差异处：

相同点：

  1. 队列和栈都是线性数据结构，即它们的元素按照一定的顺序排列。
  2. 队列和栈都可以用数组或链表来实现。

不同点：

  1. 队列中的元素先进先出，栈中的元素后进先出。
  2. 队列可以有循环队列（即循环形式），而栈只有单向的形式。
