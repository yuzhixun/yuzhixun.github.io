2.2

  1. 在顺序表中插入或删除一个元素，需要平均移动（n+1)/2元素，具体移动的元素个数与线性表中元素的个数有关。
  2. 顺序表中逻辑上相邻的元素的物理位置一定紧邻。单链表中逻辑上相邻的元素物理位置**不一定**紧邻
  3. 在单链表中，除了首元结点外，任一节点的存储位置由**指针域**指示
  4. 在单链表中设置头结点的作用是**简化链表操作、提供链表入口、处理空链表和存储附加信息**。

  1. 删除p结点的直接后继结点的语句序列是**（14）（3）（14）**。
  2. 删除p结点的直接前驱结点的语句序列是**（10）（12）（8）（11）（3）（14）**。
  3. 删除p结点的语句序列是**（10）（12）（7）（3）（14）**。
  4. 删除首元结点的语句序列是**（12）（11）（3）（14）**。
  5. 删除尾元结点的语句序列是**（9）（11）（3）（14）或者（12）（10）（6）（3）（14）**。

2.21

试写一算法，实现顺序表的就地逆置，及利用元表的存储空间将线性表（a1,a2,….,an)逆置为(an,an-1……a1)

void Change(List L,ElemType &e){
    int length = len(L) - 1;
    for(int i = 0;i<=length/2;i++){
        e = L[0];
        L[i] = L[length - 1 - i];
        L[length - 1 - i] = e;
    }
}

2.22 试写一算法，实现单链表的就地逆置

struct ListNode* reverseLinkedList(struct ListNode* head) {
    if (head == NULL || head->next == NULL)
        return head;
    
    struct ListNode* prev = NULL;
    struct ListNode* curr = head;
    
    while (curr != NULL) {
        struct ListNode* temp = curr->next;
        curr->next = prev;
        prev = curr;
        curr = temp;
    }
    
    return prev;
}
