表格的css属性

在CSS中，可以使用以下属性来控制表格的样式和布局：

  1. border-collapse：设置表格边框的合并方式。常用的取值有：
    • collapse：边框会合并为单一的边框，相邻单元格间没有间隙。
    • separate：每个单元格都有独立的边框，相邻单元格间有间隙。
  2. border-spacing：设置相邻单元格之间的边框间距。只在border-collapse属性值为separate时生效。
  3. caption-side：设置表格标题（<caption>标签）的位置。常用的取值有：
    • top：表格标题显示在表格上方。
    • bottom：表格标题显示在表格下方。
  4. empty-cells：设置空单元格（没有内容）的边框显示方式。常用的取值有：
    • show：显示空单元格的边框。
    • hide：隐藏空单元格的边框。
  5. table-layout：设置表格的布局算法。常用的取值有：
    • auto：浏览器根据内容自动调整列宽和表格布局。
    • fixed：固定表格布局，列宽由开发者手动设置。
  6. text-align：设置单元格中内容的水平对齐方式。常用的取值有：
    • left：左对齐。
    • center：居中对齐。
    • right：右对齐。
  7. vertical-align：设置单元格中内容的垂直对齐方式。常用的取值有：
    • top：顶部对齐。
    • middle：居中对齐。
    • bottom：底部对齐。
  8. width：设置表格的宽度。可以使用像素、百分比或其他长度单位进行设置。

这些属性可以应用于 <table> 元素本身，也可以应用于 <th>（表头）和 <td>（表格数据）这些单元格元素。需要注意的是，不同浏览器可能对这些属性的解析和显示有所差异。

transform的用法

在 CSS 中，transform 是一个属性，用于对元素进行变换（transformation）操作。它可以实现旋转、缩放、平移和倾斜等效果，为元素提供了强大的视觉变换能力。

transform 属性可以与不同的函数一起使用，以达到所需的变换效果。以下是常用的 transform 函数：

  1. translate()：平移元素，接受参数为长度值或百分比。
    • translateX()：水平平移。
    • translateY()：垂直平移。
    • translateZ()：三维环境下的平移。
  2. scale()：缩放元素，接受参数为数字。
    • scaleX()：水平缩放。
    • scaleY()：垂直缩放。
    • scaleZ()：三维环境下的缩放。
  3. rotate()：旋转元素，接受参数为角度值。
    • rotateX()：绕 X 轴旋转。
    • rotateY()：绕 Y 轴旋转。
    • rotateZ()：绕 Z 轴旋转。
  4. skew()：倾斜元素，接受参数为角度值。
    • skewX()：在 X 轴方向上倾斜。
    • skewY()：在 Y 轴方向上倾斜。
  5. matrix()：通过定义矩阵变换来实现更复杂的变换。

linear-gradient() 是一个 CSS 函数，用于创建线性渐变背景。它允许你在元素的背景中使用两个或多个颜色，并在它们之间创建平滑的过渡效果。

background: linear-gradient(to bottom, #f1f4f9, #dff1ff);

  • direction：指定渐变的方向，可以是关键字（如 to bottom、to top、to left、to right）或角度值（如 45deg、90deg）。方向关键字表示从哪个方向开始渐变，而角度值表示从元素中心开始的渐变角度。
  • color-stop1、color-stop2：指定颜色和位置。颜色可以是十六进制值、RGB 值、颜色关键字等。位置是可选的，表示颜色在渐变中的位置，默认按照颜色停止点的顺序均匀分布。

DOM

className

className 是 DOM 元素对象的一个属性，用于获取或设置元素的类名。

通过访问元素的 className 属性，可以获取该元素当前设置的所有类名，多个类名之间通过空格分隔。例如，如果一个元素的类名是 "class1 class2"，那么通过 element.className 获取到的值就是 "class1 class2"。

同时，也可以使用 className 属性来设置元素的类名。你可以直接给 className 赋值一个字符串，用于替换元素原有的类名。例如，element.className = "newClass" 将会将元素的类名设置为 “newClass”。

需要注意的是，使用 className 替换元素的类名时会覆盖原有的类名。如果你希望保留原有的类名，并添加新的类名，可以使用其他方法，例如使用 classList.add() 方法来添加类名。

总结起来，className 是用于获取和设置元素类名的属性。

newRow.innerHTML

newRow.innerHTML 是一个属性，用于获取或设置 HTML 元素的内容。

对于一个元素对象 newRow，newRow.innerHTML 可以返回该元素包含的 HTML 内容，以字符串的形式表示。这包括元素标签内的文本内容以及任何子元素和它们的内容。

例如，如果 newRow 是一个 <tr> 元素，那么 newRow.innerHTML 将返回该表格行内包含的 HTML 内容，包括其单元格（<td>）等子元素的 HTML 内容。

同时，你也可以使用 newRow.innerHTML 来设置元素的 HTML 内容。通过给 newRow.innerHTML 赋值一个包含有效的 HTML 代码的字符串，可以替换元素的当前内容并更新显示。

需要注意的是，使用 innerHTML 属性设置内容时，要谨慎防止跨站脚本攻击（XSS），确保只插入可信任的 HTML 内容。

总结一下，newRow.innerHTML 属性用于获取或设置 HTML 元素的内容。
