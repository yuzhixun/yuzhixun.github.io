The First C++ Program

#include <iostream>
using namespace std;

int main(){
    cout << "!!!Hello World!!! I'm "<<18<<" Today "<< endl;
    return 0;
}

#include <iostream>
using namespace std;

int main(){
    int number;
    cin >> number;
    cout << "!!!Hello World!!! I'm "<<18<<" Today "<< endl;
    cout << "!!!Hello My name is " << number;
    return 0;
}

输入：

  • 在C中，通常使用scanf函数进行输入。例如，scanf("%d", &number)可以读取一个整数并将其存储到number变量中。
  • 在C++中，通常使用cin对象进行输入。例如，cin >> number可以实现与上述相同的功能。

输出：

  • 在C中，通常使用printf函数进行输出。例如，printf("Hello World\n")可以打印出”Hello World”。
  • 在C++中，通常使用cout对象进行输出。例如，cout << "Hello World" << endl可以实现与上述相同的功能。

需要注意，整型的输出不需要加引号

什么是C++标准模板库（STL）？

​ 标准模板库 STL（Standard Template Library），是 C++ 标准库的一部分，不需要单独安装，只需要#include 头文件。

STL六大部件

  • 容器（Containers）
  • 分配器（Allocators）
  • 算法（Algorithm）
  • 迭代器（Iterators）
  • 适配器（Adapters）
  • 仿函数（Functors）

vector容器

  • vector时将元素置于一个动态数组中加以管理的容器。
  • vector可以随机存取元素（支持索引值直接存取，用【】操作符或at【】方法，这个等会详讲。
  • vector尾部添加或移除元素非常快速，但是在中部或者头部插入或者删除元素比较费劲。

#include <iostream>
#include <vector> 

using namespace std;

int main(){
    int arr[] = {1,2,3,4,5};
    vector<int> v1(arr,arr+5);
    
    for(int i = 0;i<5;i++){
        cout << v1[i] << " ";
    }
    
    cout << endl;
    
    vector<int> v2(3,10);
    for(int i = 0;i<3;i++){
        cout << v2[i] << " ";
    }
    cout << endl;
    
    vector<int> v3(v1);
    for(int i = 0;i<5;i++){
        cout << v3[i] << " ";
    }
    cout << endl;
    
    vector<int> v4 = v1;
    for(int i = 0;i<5;i++){
        cout << v4[i] << " ";
    }
    
    
    return 0;
}
