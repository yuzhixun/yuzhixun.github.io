The First C++ Program

    cout << "!!!Hello World!!! I'm "<<18<<" Today "<< endl;

#include <iostream>
using namespace std;

int main(){
    int number;
    cin >> number;
    cout << "!!!Hello World!!! I'm "<<18<<" Today "<< endl;
    cout << "!!!Hello My name is " << number;
    return 0;
}

输入：

  • 在C中，通常使用scanf函数进行输入。例如，scanf("%d", &number)可以读取一个整数并将其存储到number变量中。
  • 在C++中，通常使用cin对象进行输入。例如，cin >> number可以实现与上述相同的功能。

输出：

  • 在C中，通常使用printf函数进行输出。例如，printf("Hello World\n")可以打印出”Hello World”。
  • 在C++中，通常使用cout对象进行输出。例如，cout << "Hello World" << endl可以实现与上述相同的功能。

需要注意，整型的输出不需要加引号

stl(c++)

什么是C++标准模板库（STL）？

​ 标准模板库 STL（Standard Template Library），是 C++ 标准库的一部分，不需要单独安装，只需要#include 头文件。

STL六大部件

  • 容器（Containers）
  • 分配器（Allocators）
  • 算法（Algorithm）
  • 迭代器（Iterators）
  • 适配器（Adapters）
  • 仿函数（Functors）

vector容器

  • vector时将元素置于一个动态数组中加以管理的容器。
  • bvector可以随机存取元素（支持索引值直接存取，用【】操作符或at【】方法，这个等会详讲。
  • vector尾部添加或移除元素非常快速，但是在中部或者头部插入或者删除元素比较费劲。

数组.begin,将指针指向数组的第一个元素。
数组.end，将指针指向最后一个元素的下一个元素。

我们通过对指针的灵活运用，来更改数据的储存

#include <iostream>
#include <vector> 

using namespace std;

int main(){
    int arr[] = {1,2,3,4,5};
    vector<int>  vecINTA,vecINTB;
    vecINTA.assign(arr,arr+5);
    
    //将vecINTSA的内容赋值给vecINTB
    
    vecINTB.assign(vecINTA.begin()+2,vecINTA.end()-1); 
    
    for(int i = 0;i<vecINTA.size();i++){
        cout << vecINTA[i] <<" ";
    }
    cout << endl;
    
    for(int i = 0;i<vecINTB.size();i++){
        cout << vecINTB[i] <<" ";
    }
    
    cout << endl;
    
    
    return 0;

}

元素的互换

vector.swap

#include <iostream>
#include <vector> 

using namespace std;

int main(){

    
    int arr[] = {1,2,3,4,5};
    vector<int>  vecINTA,vecINTB,vecINTC;
    vecINTA.assign(arr,arr+5);
    
    //将vecINTSA的内容赋值给vecINTB
    
    vecINTB.assign(vecINTA.begin()+2,vecINTA.end()-1); 
    vecINTC.assign(4,10);
    
    for(int i = 0;i<vecINTA.size();i++){
        cout << vecINTA[i] <<" ";
    }
    cout << endl;
    
    for(int i = 0;i<vecINTB.size();i++){
        cout << vecINTB[i] <<" ";
    }
    cout << endl;
    
    for(int i = 0;i<vecINTC.size();i++){
        cout << vecINTC[i] <<" ";
    }
    cout << endl;
    
    cout << "交换过后" << endl;
    
    vecINTB.swap(vecINTC);
    for(int i = 0;i<vecINTB.size();i++){
        cout << vecINTB[i] <<" ";
    }
    cout << endl;
    
    for(int i = 0;i<vecINTC.size();i++){
        cout << vecINTC[i] <<" ";
    }
    cout << endl;
    return 0;
}

vecINTB.swap(vecINTC);

其中，就这一行代码，我们将vecINTB与vecINTC中的元素互换。

迭代器

什么是迭代器？

  • 迭代器是一种检查容器内元素并且遍历容器内元素的数据类型

迭代器的作用

  • 迭代器提供一个容器中的对象的访问方法，并且定义了容器中对象的范围

vector容器的迭代器

  • vector<int>::iterator iter;//变量名为iter

begin和end函数

vector::iterator it;

it是指针，并且begin和end得到的也是地址

#include <iostream>
#include <vector> 

using namespace std;

int main(){
    vector<int>::iterator it;
    int arr[] = {12,15,19,45,49};
    
    vector<int> vecintA;
    vecintA.assign(arr,arr+5);
    
    it = vecintA.begin();
    
    for(it = vecintA.begin();it!=vecintA.end()-2;it++){
        cout << *it << " ";
    }
    
        
    return 0;
}
