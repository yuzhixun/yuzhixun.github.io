**print(sum(list(map(int,input().split()))))**：

  1. input() 函数用于从控制台读取一行输入，返回的结果是一个字符串；
  2. input().split() 方法用于将这个字符串按照空格分割成一个字符串列表；
  3. map(int, ...) 函数用于将这个字符串列表中的每个元素转换为整数类型，返回的结果是一个迭代器；
  4. list(map(int, ...)) 将上一步中得到的迭代器转换为列表；
  5. sum(...) 函数用于计算列表中所有元素的和；
  6. print(...) 函数用于将上一步得到的结果输出到控制台。

因此，当输入例如 “1 2 3 4” 后，代码会将其转换为整数列表 [1, 2, 3, 4] 并计算它们的和，最终输出 10。

good = []

这是一个 Python 代码段，它定义了一个名为 goods 的空列表。

在 Python 中，列表是一种序列类型的数据结构，可以用于存储多个数据元素。列表中可以存储任何类型的数据，包括数字、字符串、元组、字典等。列表使用方括号 [] 表示，元素之间用逗号 , 分隔开。

本代码段中，goods = [] 定义了一个空列表 goods，其中没有任何元素。可以通过向 goods 列表中添加元素来实现对列表的操作，例如：

goods = []
goods.append("apple")
goods.append("banana")
goods.append("orange")
print(goods)
["apple", "banana", "orange"]

append 用在列表后面给列表添加新的元素：list.append(else)

range()函数的用法

my_list = [1, 2, 3, 4, 5, 6]
for i in range(len(my_list) - 2, 1, -1):
    if my_list[i] % 2 == 0:
        del my_list[i]
  1. 给出了一个列表
  2. 这里的range(a,b,-1)，意思是i的逆索引，i会从python的索引方式开始索引，从a开始，一直到1，默认间距-1，如上述for i in range(len(my_list) - 2, 1, -1):就是[5，4，3，2]

在返回函数之的时候，会遇到小无线循环的小数，我们可以用两种解法：一般我们选用第二种，函数走第二种，非函数走第一张。

import math
def ganweihao_shi_wo_die(r):
    return math.pi * r**2
s = ganweihao_shi_wo_die(1)
print("圆的面积为：{:.3f}".format(s))

import math
def ganweihao_shi_wo_die(r):
    return round(math.pi * r ** 2,2)
print("圆的面积为：",ganweihao_shi_wo_die(1))

isalpha()

isalpha() 是 Python 字符串的一个方法，用于判断字符串中所有的字符是否都为字母。如果字符串中所有的字符都是字母，则返回 True，否则返回 False。

isdigit()

isdigit() 是 Python 字符串的一个方法，用于判断字符串中所有的字符是否都为数字。如果字符串中所有的字符都是数字，则返回 True，否则返回 False。

isspace()

isspace() 是 Python 字符串的一个方法，用于判断字符串中所有的字符是否都是空格。如果字符串中所有的字符都是空格，则返回 True，否则返回 False。
