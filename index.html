一棵度为2的树与一棵二叉树有何区别？

解答：度为2的树是一种通用的树结构，其中每个节点可以有0、1或2个子节点。而二叉树则是一种特殊的树结构，每个节点最多只能有两个子节点，分别称为左子节点和右子节点，并且子节点的顺序是有限制的（左子节点在前，右子节点在后）。

需要注意的是，度为2的树可以包含二叉树，但二叉树不一定是度为2的树。

一.实验目的

​ 本实验旨在通过使用==pandas==库中的各种函数，对数据进行清理、分析和可视化，以理解数据中的信息，并处理缺失值、异常值，以及进行数据类型转换等操作。

二.实验步骤

  1. 导入必要的库

    • import pandas as pd
    • import numpy as np
    • from numpy import nan as NA
    • import matplotlib.pyplot as plt

  2. 利用isnull函数检测缺失值

    • df = pd.read_csv(“data.csv”) # 读取数据
    • missing_values = df.isnull() # 检测缺失值

​

  1. 解决Series中的Node值处理

  2. 统计缺失值

    • 统计每个字段的缺失值数量：missing_counts = missing_values.sum() # 计算缺失值的总数

  3. 利用info函数查看缺失值

  4. 利用dropna处理缺失值

    ==dropna()==是pandas中用于处理缺失值的函数之一，它可以从DataFrame中删除包含缺失值的行或列。

  5. 展示dropna函数axis, thresh,how参数的应用

    • axis：指定删除缺失值的轴，默认值为0，表示删除包含缺失值的行；设置为1时表示删除包含缺失值的列。
    • how：指定删除缺失值的方式，默认值为'any'，表示删除包含任何缺失值的行或列；设置为'all'时表示只有在整行或整列都是缺失值时才删除。
    • thresh参数则用于指定删除行或列的最小非空值数量

  6. 利用series的均值填充method函数
    在这个实验中，我们将使用Series的mean()函数来计算每列的平均值，然后使用fillna()函数将缺失值填充为该列的平均值。

    # 计算每列的平均值

    mean_values = df.mean(axis=0)

    # 将缺失值填充为该列的平均值

    df = df.fillna(mean_values)

  7. 展示fillna函数method的应用
    df = df.fillna(method=’ffill’) # 用前一个非空值填充
    我们不难发现，这种用法已经慢慢被淘汰，所以很多用pycharm的朋友，会发现这里不停的报错

​ 最新用法
​ ==df.ffill()==

  1. 利用fillna填充缺失值
  • fillna函数用于填充缺失值。例如：我们可以使用0填充所有的缺失值：
    df = df.fillna(0) # 用0填充所有缺失值

一下例题中我们所表示的意思是，第一列我们用0.88来填充，第二列我们用0.66来填充

  1. 用replace替换数据值

    在这个实验中，我们将使用replace()函数将特定值替换为其他值。例如，我们将所有的‘ ’值替换为0。

    df = df.replace(‘ ‘, 0)

    如果你需要替换多个特定的值，你可以传入一个字典，其中键是你要替换的值，值是替换后的值。例如，如果你想将’A’替换为’a’，将’B’替换为’b’，可以如下操作：

    df = df.replace({‘A’: ‘a’, ‘B’: ‘b’})

    这里我们将1替换为男，0替换为女

  2. 传入列表实现多值替换

    在这个实验中，我们将使用一个列表来替换多个特定的值。例如，我们将所有的[1,2]值替换为[3,4]，也就是说替换为3，2替换额为4

    df = df.replace([1,2], [3,4])

  3. map（）映射数据

    在这个实验中，我们将使用map()函数将每个元素映射到一个新的值。例如，我们将所有的字符串列中的’A’和’B’映射为’a’和’b’。

    df = df.map({‘A’: ‘a’, ‘B’: ‘b’})

三.总结

isnull()函数应用于Pandas Series或DataFrame对象来检测缺失值，它返回一个与原始数据结构相同形状的布尔值数组，其中True表示对应位置的值为缺失值（NaN），False表示对应位置的值不是缺失值。

dropna()函数是Pandas库中用于删除缺失值的函数。它可以应用于DataFrame和Series对象。在DataFrame对象上使用dropna()函数会删除包含缺失值（NaN）的行或列，具体取决于指定的轴参数。在Series对象上使用dropna()函数会删除含有缺失值的元素。

fillna()函数是Pandas库中用于填充缺失值的函数。它可以应用于DataFrame和Series对象。在DataFrame对象上使用fillna()函数可以填充包含缺失值（NaN）的单元格。在Series对象上使用fillna()函数可以填充含有缺失值的元素。

replace()函数是Pandas库中用于替换值的函数。它可以应用于DataFrame和Series对象。在DataFrame对象上使用replace()函数可以将指定的值替换为新的值。在Series对象上使用replace()函数可以将指定的值替换为新的值。

map()函数是Pandas库中用于对Series对象进行映射操作的函数。它可以将Series中的每个元素按照指定的映射关系进行替换或转换。

综上所述，这些数据处理和分析方法可以帮助我们更好地理解和处理数据，为进一步的数据分析和建模提供有效的支持。
